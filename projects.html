Car Price Prediction Web App

Built an interactive Streamlit web app to predict car prices using Linear Regression and Random Forest models. Features include CSV upload, dynamic target selection, model training, error metric evaluation (MAE, MSE, RMSE), and actual vs predicted price visualization.
Technologies used: Python, Streamlit, scikit-learn, pandas, matplotlib.

AI Resume Screener using Gradio and Python

Developed an AI-powered resume screening tool that analyzes candidate resumes against job descriptions to calculate a match score, detect available and missing skills, and visualize skill coverage using pie charts.
Technologies used: Python, Gradio, Regex, Matplotlib, NumPy.
